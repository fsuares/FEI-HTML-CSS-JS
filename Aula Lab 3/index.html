HTML e CSS

TML é a única linguagem que o navegador consegue interpretar para a exibição do conteúdo. Este paragrafo foi escrito com fonto "Comic Sans MS", letras em tamanho 25pt na cor vermelha

O HTML é um conjunto de tags responsáveis pela marcação do conteúdo de uma pagina no navegador. Diversas tags são disponibilizadas pela imagem HTML e cada uma possui uma funcionalidade especifica. Este paragrafo foi escrito com fonte "Arial Black", letras em tamanho 20pt na cor verde

CSS é uma outra linguagem, separada do HTM, com objetivo único de cuidar da estilização da página. A vantagem é que o CSS é bem mais robusto que o HTML para estilização. Com CSS, regras de estilo não aparecem mais no HTML. Este paragrafo foi escrito com fonte "Freestyle Script", letras em tamanho 40pt na cor branca.
